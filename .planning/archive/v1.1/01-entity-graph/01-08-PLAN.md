3. Build the DOM scaffolding (use template strings via concatenation):

           </svg>
           <div class="graph-nodes" style="position:relative; width:${maxX}px; height:${maxY}px;"></div>
         </div>
       </div>

For each edge in layout.edges, iterate `(edge.sections || []).forEach(section => append a <path d="..." class="graph-edge" stroke="currentColor" fill="none" stroke-width="1.5" /> to .graph-edges)`. Build the d-attribute via buildEdgePath(section). Default stroke color via `text-gray-400` on a wrapping group is acceptable; Plan 09 sets the actual stroke styling.
5. For each node in layout.children, append a <div class="graph-node" data-iri="${node.id}"> with inline style `position:absolute; left:${node.x}px; top:${node.y}px; width:${node.width}px; height:${node.height}px;`. Wrap the DOM mount in `requestAnimationFrame` to defer until the panel is visible (Pitfall 6):

        if (!graphData || !graphData.layout) return;

            _mountGraph(graphData);
        });
    }

`_mountGraph` does the actual DOM construction.

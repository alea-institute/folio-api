    .graph-edge {
        stroke: #9CA3AF;            /* Tailwind gray-400 — UI-SPEC color contract */
        fill: none;
        stroke-width: 1.5;
    }

    .graph-node {
        /* base layer: positioning is inline; cursor + transitions only */
        transition: box-shadow 0.12s ease-out;
        user-select: none;
    }

    /* Selected node glow — UI-SPEC §Color row 2 (D-20). */
    .graph-node-selected > div {
        box-shadow: 0 0 12px rgba(37, 99, 235, 0.35);

    /* Reduced-motion: no transition delay on selection */
    @media (prefers-reduced-motion: reduce) {
        .graph-node {
            transition: none;
        }
    }

The `> div` selector targets the inner styled wrapper (the one with `bg-white border-gray-300 rounded`), since that's where the visual border lives. The 12 px blur and the exact rgba match UI-SPEC §Color (D-20).
